## Positioning

Claude Code gives you one answer. `founder-os` lets you ask multiple agents, compare consensus, surface disagreement, and synthesize the useful part.

Short tagline:

> Multi-agent primitives for Claude Code: consensus, debate, fanout research, skill building, and measured optimization.

## Submission Copy

Short:

> founder-os is a Claude Code plugin that adds reusable multi-agent coordination primitives: stochastic consensus, debate rooms, fanout research, skill building, and measured optimization loops.

Long:

> founder-os is a Claude Code plugin for running multi-agent workflows inside Claude Code. The core primitive is `/stochastic`: poll N agents with the same prompt, then aggregate consensus, divergences, and outliers. It also includes `/model-chat` for debate-room synthesis, `/fanout` for parallel research, `/skillbuilder` for creating stronger Claude Code skills, and `/autoresearch` for measured hill-climb optimization loops.
