# Four-Lens Feature Ideation

Four lenses, one grid. Each catches features the others miss.

## Lens 1: Job-Backward

Start from each job → brainstorm features that deliver it.

**Question**: "What would we build to make this job trivially easy?"

Produces: depth — multiple approaches per job.

## Lens 2: Customer Journey

Map the timeline → ideate at each stage.

**Stages**: Discover → Sign up → First week → Month 1 → Month 3 → Month 6+

**Question**: "What does the customer need at this exact moment?"

Produces: onboarding and retention features that job-backward misses.

## Lens 3: Competitive Displacement

List every tool they currently pay for → ideate replacements.

**Question**: "What lets them cancel [tool]?"

Produces: "cancel everything else" positioning + concrete displacement math for pricing.

## Lens 4: Value Chain

Map the full chain → ideate at each link.

Adapt the chain per product. Example:
```
Generation → Capture → Conversion → Delivery → Retention → Referral
```

**Question**: "Where does value leak out?"

Produces: full-funnel coverage, reveals gaps between stages.

## The Master Grid

Combine all four into one grid:

```
              | Chain Stage 1 | Stage 2 | Stage 3 | Stage 4 | Stage 5 | Stage 6 |
Job 1         | [features]    | ...     | ...     | ...     | ...     | ...     |
Job 2         | ...           | ...     | ...     | ...     | ...     | ...     |
```

Each cell: feature name + journey stage + what it displaces.

Empty cells are fine. Full cells at every intersection = over-building.

## After the Grid

1. **Circle must-haves** — features appearing across multiple lenses
2. **Kill nice-to-haves** — single-cell features with low job priority
3. **Group into tiers** — Tier 1 = top 1-2 jobs. Tier 2 = expand. Tier 3 = lock in.
4. **Check displacement math** — each tier displaces more cost than it charges
